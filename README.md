# TradeMate

TradeMate is a full-stack trading journal and risk management platform built to help traders record trades, track performance, analyze mistakes, and improve trading discipline.

The project also serves as a practical learning platform for backend engineering using Go, PostgreSQL, authentication, APIs, clean architecture, and mobile development.

---

## Problem Statement

Many beginner traders lose consistency because they do not properly track their trades, emotions, risk management, and performance.

TradeMate aims to solve this problem by providing:

* Trade Journaling
* Risk Management Tools
* Performance Analytics
* Emotion Tracking
* Mistake Analysis
* Weekly and Monthly Reviews
