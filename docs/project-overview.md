# TradeMate - Project Overview

## What is TradeMate?

TradeMate is a full-stack trading journal and risk management application being built to combine software engineering and trading knowledge into a real-world product.

The application allows traders to record trades, calculate risk, analyze performance, track emotions, identify mistakes, and improve trading discipline over time.

The project is also being used as a learning platform to develop professional backend engineering skills using Go, PostgreSQL, authentication, APIs, and mobile development.

---

# Problem Statement

Many beginner traders lose consistency because they do not properly track their trading decisions.

Common problems include:

* No structured trade journal
* No performance tracking
* No risk management records
* No win rate statistics
* No mistake tracking
* No emotional analysis
* No weekly or monthly review process

As a result, traders often repeat the same mistakes without realizing it.

TradeMate aims to solve these problems by creating a structured system for recording and analyzing trading activity.
